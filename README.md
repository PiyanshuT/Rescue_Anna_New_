# Rescue_Anna_New_
Rescue Anna (derived from Annapurna — the goddess of nourishment) is an Android application designed to bridge the gap between excess food and empty stomachs. This platform allows users to effortlessly donate surplus food or receive donations, reducing food waste and fighting hunger in local communities.

✨ Key Features
User Authentication: Secure Sign-up and Login for donors and receivers.
Donate Food: Easily create listings with food type, quantity, and address.
Receive Food: Browse available food listings and accept donations.
History Check: Track past donations and receive notifications.
User-friendly Interface: Intuitive design for quick, efficient actions.
